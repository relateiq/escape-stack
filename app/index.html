Escape Stack

Escape Stack provides you a sweet way of handling things that close (or do other things) on escape.

Fancy boxes (boxes with dashed borders) each have an escape handler attached that removes the box when fired.
Whenever the escape key is pressed, the handler is called and is popped off the stack.
You can set the escape stack to pop each handler one at a time or all of them at once (and technically anything in between).
